## 📱 หน้าจอทั้งหมด (Sitemap)

```
Panya App
├── 🔓 Auth
│   ├── /login
│   └── /register
│
├── 🚀 Onboarding (3 steps)
│   └── /onboarding
│
├── 📊 Dashboard (หลัง login)
│   ├── / (Home — Daily Briefing + Quick Stats)
│   ├── /menu (จัดการเมนู)
│   ├── /knowledge (ศูนย์ความรู้ AI)
│   ├── /chat-logs (บทสนทนา + Learning Loop)
│   ├── /content (สร้างโพสต์ AI)
│   ├── /analytics (ยอดขาย + Forecast)
│   ├── /customers (จัดการลูกค้า)
│   └── /settings (ตั้งค่าร้าน + AI Style)
│
└── 🌐 Public
    └── /menu/:shopId (เมนูสาธารณะ)
```

```

```

## 🎨 Design System อ้างอิง

### Colors
| ชื่อ | Hex | ใช้สำหรับ |
|------|-----|----------|
| Primary | `#1A237E` | ปุ่มหลัก, header, logo |
| Secondary | `#00B4D8` | links, accents, icons |
| Amber | `#F4A261` | warnings, highlights |
| Dark | `#1A1A2E` | ข้อความหลัก |
| Light | `#F8FAFE` | background |
| Success | `#2D9C5A` | positive metrics, ✅ |
| Gradient | `#1A237E→#00B4D8` | hero, logo, CTA |

### Fonts
| ประเภท | Font | ใช้สำหรับ |
|--------|------|----------|
| Display | Prompt 700 | Logo, หัวข้อใหญ่ |
| Heading | Prompt 600 | Section headers |
| Body TH | Sarabun 400 | เนื้อหาภาษาไทย |
| Body EN | Inter 400 | เนื้อหาภาษาอังกฤษ |

### Design Principles
1. **เรียบง่าย** — เจ้าของร้านไม่ใช่ tech person ต้องใช้ง่าย
2. **Mobile-first** — ดูบน LINE in-app browser ได้ดี
3. **ข้อมูลชัดเจน** — ตัวเลขสำคัญต้อง bold, ใช้สี distinguish
4. **AI เป็นพระเอก** — ทุกหน้าต้องแสดงให้เห็นว่า AI ช่วยอะไร
5. **ภาษาไทย** — UI ทั้งหมดเป็นภาษาไทย
